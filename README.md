# Match 3 - Element Battle Game

A match-3 puzzle game with elemental combat mechanics, chests, and treasure collection!

- **Battle System**: Two monsters battle based on your matches
- **Streak Multipliers**: Chain matches of the same element for bonus damage
- **Chest System**: Magical chests spawn randomly and open when you match your ally's element
- **Treasure Collection**: Collect gems, coins, stars, and crystals
- **Beautiful Animations**: Smooth orb animations, falling effects, and visual feedback

## How to Play

1. Click and drag to swap adjacent orbs
2. Match 3 or more of the same element to score
3. Match your **ally's element** to attack the enemy and open chests
4. Avoid matching the **enemy's element** or they'll attack you
5. Build streaks by matching the same element consecutively for bonus damage
6. Collect treasure from opened chests!

## Game Mechanics

- **Player Moves**: Only matches from your swaps count toward streaks
- **Cascade Matches**: Automatic matches from falling orbs deal damage but don't affect streaks
- **Chest Spawning**: 15% chance to spawn after each match
- **Chest Opening**: Match your ally's element to open all unopened chests
- **Loot**: Each chest contains 1-3 random treasure items
